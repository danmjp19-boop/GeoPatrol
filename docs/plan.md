# GeoPatrol

## Objetivo

Registrar y supervisar recorridos de patrullaje mediante GPS en tiempo real.

## Versión 1

- Login
- Inicio de turno
- Fin de turno
- Mapa GPS
- Historial de recorridos

## Futuro

- Cobertura de cuadrantes
- Supervisor en tiempo real
- Reportes PDF
- Planes de suscripción
